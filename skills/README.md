# Skills

Skills are reusable writing/operation modules HarunAI can apply on request.

## How to use
- Create a new markdown file in this folder, e.g. `skills/proposal-writing.md`.
- In chat, tell HarunAI: “Use `skills/proposal-writing.md`”.

## What to put in a skill
- Preferred structure (headings, sections)
- Tone (formal, concise, technical)
- Checklists (what must be included)
