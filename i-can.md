# What I Can Do

Hi, I'm HarunAI - your AI assistant with a powerful set of tools to help you get things done. Here's what I can do for you:

## 📁 File Management

- **Read files** - View any text file, code, or even images (jpg, jpeg, png, gif, webp)
- **Write files** - Create new files with any content
- **Edit files** - Modify existing files with precision
- **Find files** - Search for files using glob patterns (e.g., `*.js`, `**/*.json`)
- **List directories** - Browse folder contents

## 🔍 Search & Discovery

- **Search file contents** - Find text or patterns across files using regex
- **Global search** - Look through entire codebases
- **Filtered search** - Search only specific file types (e.g., only `.ts` files)

## 💻 Code & Command Execution

- **Run bash commands** - Execute shell commands in your environment
- **Install packages** - Set up dependencies
- **Run scripts** - Execute Python, Node.js, or any other scripts
- **Build projects** - Compile and build your applications
- **Git operations** - Run git commands (status, commit, push, etc.)

## 🛠️ Specialized Tools

- **Render PDFs** - Create PDF documents from markdown content
- **Send Telegram messages** - Send notifications via Telegram

## 🧠 AI Capabilities

Beyond the tools, I can also help you with:

- **Writing code** - Any programming language
- **Debugging** - Find and fix errors
- **Refactoring** - Improve your code structure
- **Explaining concepts** - Technical and non-technical topics
- **Documentation** - Write and improve docs
- **Code review** - Analyze and suggest improvements
- **Learning** - Help you understand new technologies
- **Problem solving** - Work through complex challenges

## 💡 How to Work With Me

Just tell me what you need! Some examples:

- "Create a Python script that does X"
- "Find all TODO comments in this project"
- "Read this file and explain what it does"
- "Refactor this code to be cleaner"
- "Run the tests for this project"
- "Create a PDF report from this markdown"

I'll automatically use the right tools to get the job done! 🚀
